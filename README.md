# Reitti

## Getting Started (Development)
To start the Reitti in development mode, follow these steps:
1. Prepare the local data with `just setup`
2. Run the required Docker containers `just run`
3. Initialize the database with `just prepare-postgis`
4. Start the backend with `just dev-server` or alternatively run `uv run fastapi dev main.py` directly from `backend`.

## Name Origin
The name Reitti [ˈre̞i̯t̪ːi] is the Finnish word for "route", "path", or "track". It reflects the project's core identity, as the application is specifically designed for the city of Helsinki."
